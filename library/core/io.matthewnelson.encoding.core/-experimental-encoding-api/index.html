annotation class ExperimentalEncodingApi(source)

Denotes an api as experimental, such that it may change inadvertently without notice.

Any usage of a declaration annotated with ExperimentalEncodingApi must be accepted by annotating that usage with the OptIn annotation, e.g @OptIn(ExperimentalEncodingApi::class), or by using the following compiler argument:

-Xopt-in=io.matthewnelson.encoding.core.ExperimentalEncodingApi
